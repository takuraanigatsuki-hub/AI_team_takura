# AI Team Room (AI_team_takura)

Виртуальная команда из 8 ИИ-агентов: PM, архитектор, backend, frontend, QA, ревьюер, документация, DevOps.

## Возможности

- 3D-студия с аватарами агентов
- Рабочий чат и отдельный чат обучения
- Планирование задач через PM Виктора
- React Preview и экспорт готовых сайтов (Соня)
- Самообучение из Wikipedia, GitHub, Хабра, Dev.to и др.

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

Откройте http://localhost:8000

## Стек

Python · FastAPI · WebSocket · Three.js
