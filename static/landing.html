    Возможности

    Всё для продуктовой команды

    От идеи до деплоя — агенты берут на себя планирование, дизайн, код, тесты, документацию и выкладку.

    🎮

    3D Студия

    Живая команда на сцене — клик по агенту открывает чат, speech bubbles и confetti при завершении задач.

    💬

    Командный чат

    Streaming LLM, @mentions, история задач и артефактов — общайтесь с командой как в Slack.

    📌

    Kanban & Sprint

    Backlog, приоритеты, спринты с burndown и статусами — PM Виктор ведёт доску автоматически.

    🎨

    Figma → React

    Sonya Studio: импорт макета, Compare (score совпадения), React Preview и Deploy ZIP.

    ⚡

    Cursor SDK

    Лео пишет код локально или через Cloud Agent — PR на GitHub и auto-sync для admin.

    📽️

    Презентации & 3D

    Ника — pitch deck и слайды, Зоя — интерактивные Three.js сцены и glTF-модели.

    🧪

    QA & Sandbox

    Playwright-автотесты UI, pytest, Docker sandbox для безопасного запуска кода.

    📊

    Dashboard

    Командный центр: задачи, pipeline, интеграции, timeline и статистика комнаты.

    🛡️

    Admin & роли

    Owner, admin, tech_admin, support, investor — гибкие привилегии и управление пользователями.

    💬

    Поддержка

    Тикеты, чат с оператором, готовые решения по темам — для пользователей и staff-панель.

    👤

    Кабинет & подписки

    Регистрация, профиль, тарифы, баланс кредитов и мастер первой настройки.

    📱

    PWA & Telegram

    Установка на телефон, push-уведомления и Telegram-бот для контроля комнаты.

    Как это работает

    Три шага до результата

    1

    Поставьте задачу

    Опишите цель в чате — PM Виктор разобьёт её на подзадачи и назначит агентов.

    2

    Наблюдайте в 3D

    Следите за прогрессом в студии: кто работает, что готово, какие артефакты появились.

    3

    Получите результат

    Код, UI, презентации и деплой — с GitHub sync и React Preview в один клик.
