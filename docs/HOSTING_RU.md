# Где зарегистрироваться и оплатить в рублях

Для AI Team Room нужны: **VPS** (Docker, 2 GB RAM) + **домен** (опционально, но удобно для HTTPS).

Все перечисленные сервисы — **российские**, оплата **₽**: карта МИР/Visa/MC российского банка, **СБП**, ЮMoney, SberPay, счёт для ИП/ООО.

> Цены ориентировочные — смотрите актуальные тарифы на сайтах.

---

## Рекомендация «всё в одном» (проще всего)

### [REG.RU](https://www.reg.ru/vps/linux) — VPS + домен + DNS

| Что | Зачем |
|-----|--------|
| VPS | Сервер для Docker |
| Домен `.ru` / `.рф` | `room.ваш-сайт.ru` |
| DNS | A-запись в той же панели |

**Тариф для проекта:** `Std C2-M2-D40` — **2 CPU, 2 GB RAM, 40 GB** (~980 ₽/мес или почасово от ~100 ₽ на балансе).

**Оплата:** карта, СБП, ЮMoney, счёт для юрлиц.  
**Панель:** понятная, на русском, подходит новичкам.

**Домен `.ru`:** от ~199–399 ₽/год в том же кабинете.

---

## Рекомендация «облако + удобство»

### [Timeweb Cloud](https://timeweb.cloud/services/vds-vps)

| Плюсы | |
|-------|--|
| Почасовая оплата | от **50 ₽** на баланс, списание по часам |
| NVMe, Docker | нормально тянет `docker-compose.prod.yml` |
| Локации | Москва, СПб, + EU/KZ при необходимости |
| Оплата | карта, **СБП**, ЮMoney, SberPay |

**Тариф:** Cloud MSK **2 GB RAM** — ориентир **~880–1062 ₽/мес** (или меньше при почасовом использовании).

Домен можно купить у REG.RU / Timeweb / nic.ru и прописать A-запись на IP Timeweb.

## Рекомендация «надёжнее, чуть сложнее»

### [Selectel](https://selectel.ru/services/cloud/servers/)

| Плюсы | |
|-------|--|
| Tier III ДЦ | Москва, Санкт-Петербург |
| Почасовой биллинг | pay-as-you-go |
| DDoS L3/L4 | в базе |
| Оплата | карта, счёт, безнал для компаний |

**Минимум для проекта:** 2 vCPU, 2 GB RAM, 20–40 GB — ориентир **~1000–1500 ₽/мес**.

Подходит, если планируете рост нагрузки и нужен «взрослый» облачный провайдер.

## Что заказать под AI Team Room

```
VPS:  Ubuntu 22.04
      2 vCPU
      2 GB RAM
      40 GB SSD/NVMe
      1 IPv4

Домен (опционально): room.ваш-бренд.ru
DNS: A @ или room → IP VPS

Оплата: пополнить баланс / оплатить месяц картой или СБП
```

---

## Пошагово после оплаты

1. В панели хостера создать VPS, выбрать **Ubuntu 22.04**.
2. Скопировать **IP** сервера.
3. В DNS домена: **A-запись** → этот IP.
4. SSH: `ssh root@IP`
5. Деплой по [WHERE_TO_CONNECT.md](./WHERE_TO_CONNECT.md).

## Что не обязательно платить отдельно

| Сервис | Нужен? |
|--------|--------|
| SSL (HTTPS) | **Нет** — бесплатно через Caddy + Let's Encrypt на VPS |
| GitHub | **Бесплатно** для публичного/приватного repo |
| Smart AIPI (LLM) | Отдельная подписка/API — не хостинг, ключ в `.env` |
